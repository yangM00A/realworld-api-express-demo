# 这个是跟随B站上的一个学习项目 express + MongoDB   


## 用前须知  *以我使用的为主*

1. 安装nodejs *v10.13.0*
2. MongoDB *v3.2.0*
3. supervisor


## 启动 命令 
1. 安装 supervisor
```
npm run dev
```
2. node直接启动 (无需安装supervisor)
```
node app.js
```

## 用到的技术和插件

1. express  
2. cors *处理跨域*  
3. express-validator *请求参数校验*
4. jsonwebtoken *JWT 生成token*
5. mongoose *连接MongoDB*
6. morgan *日志中间件*
